**** ขั้นตอนการติดตั้ง PhpMyAdmin with Apache2 ****
--- พิมพ์คำสั่ง $ sudo apt-get install php5-mcrypt phpmyadmin 
--- เลือกรายการ apache2
--- กรอกข้อมูล Password & Password Confirm ให้กับ admin mysql
--- กรอกข้อมูล Password ให้กับ user: root
--- พิมพ์คำสั่ง $ sudo service apache2 restart
--- เปิดเว็บ Browser พิมพ์ URL: http://192.168.43.128/phpmyadmin

**** เพิ่มความปลอดภัย  PhpMyAdmin ****
--- พิมพ์คำสั่ง $ sudo nano /etc/apache2/apache2.conf
--- ค้นหา [ Ctrl + w ] คำว่า ports.conf 
--- เพิ่มคำสั่ง Include ลงไปภายในไฟล์ apache2.conf
        * Example. | apache2.conf
        * Include ports.conf
        - Include /etc/phpmyadmin/apache.conf
--- พิมพ์คำสั่ง $ sudo nano /etc/phpmyadmin/apache.conf
--- เพิ่มคำสั่ง AllowOverride All ลงไปภายในไฟล์ apache.conf
        * Example. | apache.conf
        * DirectoryIndex index.php
        - AllowOverride All
--- พิมพ์คำสั่ง $ sudo nano /usr/share/phpmyadmin/.htaccess
--- เพิ่มคำสั่ง Auth ลงไปภายในไฟล์ .htaccess
        * Example. | .htaccess
        - AuthType Basic
        - AuthName "Administration"
        - AuthUserFile /etc/apache2/.phpmyadmin.htpasswd
        - Require valid-user
--- พิมพ์คำสั่ง $ sudo htpasswd -c /etc/apache2/.phpmyadmin.htpasswd admin 
--- กรอกข้อมูล Password & Password Confirm ให้กับ user: admin
--- พิมพ์คำสั่ง $ sudo service apache2 restart
--- เปิดเว็บ Browser พิมพ์ URL: http://192.168.43.128/phpmyadmin
