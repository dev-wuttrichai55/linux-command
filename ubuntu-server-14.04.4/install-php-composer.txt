========================================================================================================
Install PHP Composer
========================================================================================================

- พิมพ์คำสั่ง $ sudo apt-get install -y curl
- พิมพ์คำสั่ง $ sudo curl -s http://getcomposer.org/installer | php
- พิมพ์คำสั่ง $ sudo mv composer.phar /usr/local/bin/composer

- พิมพ์คำสั่ง $ composer
