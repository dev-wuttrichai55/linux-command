========================================================================================================
Unpack (แตกไฟล์) | Type .tgz
========================================================================================================

- พิมพ์คำสั่ง $ wget https://pecl.php.net/get/mongo-1.6.14.tgz					# ดาวโหลดไฟล์
- พิมพ์คำสั่ง $ sudo tar zxvf mongo-1.6.14.tgz									# แตกไฟล์ลงใน Directory Current

- พิมพ์คำสั่ง $ sudo tar zxvf mongo-1.6.14.tgz -C /tmp/data						# แตกไฟล์ลงใน Directory (/tmp/data)
